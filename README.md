# DCM-MultiOmics-Pipeline

An R-based bioinformatics workflow for identifying and characterising phagocytosis-related biomarkers in dilated cardiomyopathy / heart failure using bulk transcriptomics, machine learning, single-cell RNA-seq, immune infiltration analysis, pathway enrichment, regulatory network analysis, Human Protein Atlas annotation, and molecular docking / molecular dynamics visualisation.

## Project overview

This project integrates public GEO transcriptomic and single-cell datasets to investigate phagocytosis-associated molecular features in heart failure-related cardiomyopathy. The workflow first downloads and preprocesses bulk expression data, identifies differentially expressed genes, intersects them with phagocytosis-related gene sets, and then applies machine-learning models to select diagnostic candidate genes. The selected genes are further evaluated through validation datasets, immune infiltration, GSEA, single-cell annotation, macrophage/monocyte subtype analysis, CellChat cell-cell communication analysis, protein-protein interaction networks, nomogram construction, HPA annotation, and molecular docking / molecular dynamics result plotting.

The main candidate genes used in downstream modules include **FCN3**, **FCER1G**, and **MAP2K1**.

## Main features

- GEO data download and preprocessing
- Bulk RNA-seq / microarray differential expression analysis
- Phagocytosis-related candidate gene screening
- Machine-learning biomarker selection using LASSO, SVM-RFE, and random forest
- External validation using independent GEO data
- ROC analysis and diagnostic model evaluation
- GSEA and pathway enrichment analysis
- Immune infiltration estimation and immune-cell correlation analysis
- Single-cell RNA-seq processing and cell-type annotation using Seurat
- Macrophage / monocyte subtype analysis
- Cell-cycle and metabolism-related single-cell scoring
- CellChat-based cell-cell communication analysis
- PPI and regulatory network construction
- Nomogram, calibration curve, and decision curve analysis
- Human Protein Atlas annotation
- Molecular docking / molecular dynamics visualisation

## Datasets used

The scripts are designed around the following public datasets:

| Dataset | Role in this project | Description in workflow |
|---|---|---|
| GSE57338 | Training bulk transcriptomic dataset | Used for differential expression and model training |
| GSE5406 | Validation bulk transcriptomic dataset | Used for external validation of candidate genes/models |
| GSE183852 | Single-cell RNA-seq dataset | Used for single-cell annotation and cell-type/subtype analysis |

Phagocytosis-related gene sets are retrieved from MSigDB, including GO, KEGG, Reactome, and WikiPathways gene sets related to phagocytosis and Fc-gamma receptor-mediated pathways.

## Workflow

### 1. Raw data preparation

Script:

```text
scripts/r.00_rawdata.R
```

This script creates the raw-data folders, downloads GEO series matrix files, processes gene expression matrices, saves phenotype/group information, downloads the single-cell data object, and prepares phagocytosis-related gene sets from MSigDB.

Main outputs include:

```text
00_rawdata/training_GSE57338/GSE57338_gene_expression.csv
00_rawdata/training_GSE57338/GSE57338_group.csv
00_rawdata/validation_GSE5406/GSE5406_gene_expression.csv
00_rawdata/validation_GSE5406/GSE5406_metadata.csv
00_rawdata/singlecell_GSE183852/GSE183852_DCM_Cells.Robj.gz
gene_sets/Phagocytosis_all_genes_unique.csv
```

### 2. Differential expression and candidate genes

Script:

```text
scripts/r.01_deg.R
```

This script performs differential expression analysis between heart failure and control samples in the training dataset and intersects differentially expressed genes with phagocytosis-related genes.

Main outputs include:

```text
01_deg/DEG_GSE57338_full.csv
01_deg/DEG_GSE57338_significant.csv
01_deg/Candidate_genes_DEG_x_Phagocytosis.csv
01_deg/figures/
```

### 3. Machine-learning biomarker screening

Script:

```text
scripts/r.02_ml.R
```

This script uses candidate genes to build and compare machine-learning models. It includes LASSO, SVM-RFE, random forest, Venn intersection, ROC analysis, and external validation.

Main outputs include:

```text
02_ml/key_genes.txt
02_ml/ROC_plots/
02_ml/model_performance_tables/
```

### 4. Single-cell RNA-seq analysis

Scripts:

```text
scripts/r.03_single_cell.R
scripts/r.03b_monocyte_subtype.R
scripts/r.sub.R
```

These scripts process the single-cell object, perform quality control, clustering, cell-type annotation, cell-type abundance analysis, macrophage/monocyte subtype analysis, marker identification, and enrichment analysis.

Main outputs include:

```text
03_single_cell/seurat_clustered.rds
03_single_cell/seurat_annotated.rds
03_single_cell/05_celltype_summary.csv
03_single_cell/05_cluster_annotation.txt
```

### 5. Functional enrichment and GSEA

Script:

```text
scripts/r.04_GSEA.R
```

This script performs correlation analysis and gene set enrichment analysis for selected target genes.

Main outputs include:

```text
04_GSEA/GSEA_GO_*.csv
04_GSEA/GSEA_KEGG_*.csv
04_GSEA/GSEA_TOP10_*.csv
04_GSEA/GSEA_integrated_all_top10.csv
```

### 6. Immune infiltration analysis

Script:

```text
scripts/r.05_immune.R
```

This script estimates immune-cell signatures, compares immune infiltration between heart failure and control groups, and evaluates correlations between immune cells and key genes.

Main outputs include:

```text
05_immune/immune_scores_22cell_train.csv
05_immune/immune_wilcox_HFvsCtrl.csv
05_immune/immune_cell_correlation_22cell.csv
05_immune/immune_keygene_correlation_22cell.csv
```

### 7. Nomogram and diagnostic model evaluation

Script:

```text
scripts/r.09_nomogram.R
```

This script builds a nomogram based on selected genes and evaluates model calibration, ROC performance, and decision-curve analysis.

Main outputs include:

```text
06_nomogram/nomogram.png
06_nomogram/calibration.png
06_nomogram/DCA_curve.png
06_nomogram/nomogram_ROC.png
06_nomogram/nomogram_coef.csv
```

### 8. PPI, regulatory network, and chromosome visualisation

Scripts:

```text

scripts/r.10b_RCircos.R
scripts/r.11_network_v2.R
```

These scripts construct protein-protein interaction networks, regulatory networks, hub-gene tables, and chromosome location visualisations.

### 9. HPA annotation

Script:

```text
scripts/r.HPA.R
```

This script queries or summarises Human Protein Atlas information for selected candidate genes and generates expression visualisations.

Main outputs include:

```text

10_HPA/HPA_tissue_expression.png
```

### 10. CellChat, metabolism, cell cycle, and molecular docking visualisation

Scripts:

```text
scripts/r.cellchat.R
scripts/r.cellcycle.R

scripts/r.21b_metabolism_bubble.R
```

## Installation

This project is written mainly in R. A recent R version is recommended.

Example package installation:

```r
install.packages(c(
  "data.table", "dplyr", "tidyr", "stringr", "ggplot2", "ggrepel",
  "pheatmap", "RColorBrewer", "caret", "e1071", "glmnet",
  "randomForest", "pROC", "reshape2", "ggpubr", "igraph",
  "ggraph", "rms", "ResourceSelection", "httr", "jsonlite"
))

if (!requireNamespace("BiocManager", quietly = TRUE)) {
  install.packages("BiocManager")
}

BiocManager::install(c(
  "GEOquery", "Biobase", "limma", "org.Hs.eg.db", "AnnotationDbi",
  "clusterProfiler", "msigdbr", "Seurat", "AUCell", "GSEABase",
  "CellChat", "UCell", "RCircos"
))
```

Some packages may require additional system dependencies. CellChat, Seurat, and molecular docking/MD-related files may need separate installation or local input files depending on the computing environment.

## Notes

- The scripts were developed for a specific project directory and may require path modification before reuse.
- GEO datasets and MSigDB gene sets are downloaded or processed during the workflow.
- Generated RDS objects and result figures are not included in this repository because they can be large.
- Some advanced modules, including CellChat and molecular docking/MD visualisation, may require additional local files.
- Results may vary slightly depending on package versions and preprocessing settings.
